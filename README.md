# Firstie

Firstie é uma aplicação web que sorteia quem vai jogar primeiro em uma partida, com base nos nomes dos jogadores adicionados.
[Acesse o projeto](https://tamiresborota.github.io/firstie/)

![preview](preview.png)

## Como usar

Digite o nome de um jogador e clique em + para adicionar
Repita para todos os jogadores
Clique em Pick a name
O primeiro jogador aparecerá em destaque

## Tecnologias

HTML
CSS
JavaScript

## O que aprendi desenvolvendo esse projeto

Manipulação dinâmica do DOM com createElement e appendChild
Filtragem de arrays com filter e verificação com startsWith para tratar nomes duplicados
Estilização de scrollbar customizada com scrollbar-color e scrollbar-gutter
Layout de duas colunas com CSS Grid e grid-template-columns
Navegação por teclado com addEventListener('keydown')
Feedback visual com opacity e setTimeout
